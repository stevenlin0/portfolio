Electronic Circuit Simulator

A tool where you can drag and drop parts to build and test circuits. After signing up or logging in, you can start a project and use a zoomable canvas to add batteries, resistors, switches, wires, and bulbs. Each part shows real-time feedback like voltage, current, and resistance. Your projects are saved in a database so you can load or delete them later. When you connect the parts, the simulator shows how electricity flows and how components, like light bulbs, react.
